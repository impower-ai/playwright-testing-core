# Function: hasFlag()

> **hasFlag**(`value`, `flag`, `enumType`): `boolean`

Checks if a value has a specific flag.

## Parameters

• **value**: `string` \| `number`

The value to check.

• **flag**: `string` \| `number`

The flag to check for.

• **enumType**: `object`

The enum object.

## Returns

`boolean`

True if the value has the flag, false otherwise.
