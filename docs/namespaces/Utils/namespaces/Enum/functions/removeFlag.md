# Function: removeFlag()

> **removeFlag**(`value`, `flag`, `enumType`): `number`

Removes a flag from a value for a flag enum.

## Parameters

• **value**: `number`

The original value.

• **flag**: `number`

The flag to remove.

• **enumType**: `object`

The enum object.

## Returns

`number`

The new value with the removed flag.

## Throws

If the enum is not a flag enum.
