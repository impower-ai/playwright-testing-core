# Function: toggleFlag()

> **toggleFlag**(`value`, `flag`, `enumType`): `number`

Toggles a flag in a value for a flag enum.

## Parameters

• **value**: `number`

The original value.

• **flag**: `number`

The flag to toggle.

• **enumType**: `object`

The enum object.

## Returns

`number`

The new value with the toggled flag.

## Throws

If the enum is not a flag enum.
