# Function: addFlag()

> **addFlag**(`value`, `flag`, `enumType`): `number`

Adds a flag to a value for a flag enum.

## Parameters

• **value**: `number`

The original value.

• **flag**: `number`

The flag to add.

• **enumType**: `object`

The enum object.

## Returns

`number`

The new value with the added flag.

## Throws

If the enum is not a flag enum.
