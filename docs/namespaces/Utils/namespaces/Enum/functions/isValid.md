# Function: isValid()

> **isValid**(`value`, `enumType`): `boolean`

Validates if a given value is a valid enum value.

## Parameters

• **value**: `any`

The value to validate.

• **enumType**: `object`

The enum object to validate against.

## Returns

`boolean`

True if the value is valid, false otherwise.
