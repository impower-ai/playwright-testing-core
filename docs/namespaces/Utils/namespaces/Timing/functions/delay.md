# Function: delay()

> **delay**(`timeout`): `Promise`\<`void`\>

Creates a delay for a specified duration.

## Parameters

• **timeout**: `number`

The delay duration in milliseconds.

## Returns

`Promise`\<`void`\>

A Promise that resolves after the specified delay.
